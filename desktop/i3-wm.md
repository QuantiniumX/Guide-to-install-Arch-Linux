# how to install i3

~~~
$ udo pacman -Syu xorg xorg-server xorg-xinit i3-wm i3-gaps i3blocks i3lock numlockx lightdm lightdm-gtk-greeter ttf-jetbrains-mono terminius-font alacritty rofi
$ sudo systemctl enable lightdm
$ reboot
~~~
